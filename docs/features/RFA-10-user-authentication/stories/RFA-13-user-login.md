## Description:
This story enables the RepIt User to log into and log out of their account, with proper session handling and error management.

## User:
RepIt User

## Benefit Hypothesis:
As a RepIt User, I would like to log in and out of my account so that I can securely access my data and manage my session.

## Acceptance Criteria:

I will be satisfied when:
- User can log in using the following logic
- - User can log in using previously registered methods (Google, Apple, Facebook, Email)
  - Invalid login attempts display an error message
  - Error handling is in place for invalid login attempt
- User can log out successfully
- - User session is terminated upon logout
  - User is redirected after logout
- User remains logged in during active usage
- Expired or invalid sessions require re-authentication
