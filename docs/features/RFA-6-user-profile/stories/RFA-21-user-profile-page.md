## Description:
This story enables the RepIt User to view and interact with the User Profile page, which organizes all profile-related features into structured, editable regions.

## User:
RepIt User

## Benefit Hypothesis:
As a RepIt User, I would like to access all my profile information in one place so that I can easily view and manage my personal data, preferences, and records.

## Acceptance Criteria:

I will be satisfied when:
- The RepIt User can access the User Profile Page  

- The User Profile Page contains the following sections as viewable and editable regions  
- - Demographic Info  
  - Workout Preferences  
  - Personal Records  

- Each section is clearly separated and labeled  

- The RepIt User can navigate between sections within the User Profile Page  

- The RepIt User can view data within each section  

- The RepIt User can edit data within each section  

- Updates made within each section are reflected in the User Profile  

- All sections are accessible within a single unified profile page  

## Dependencies:
User Profile
