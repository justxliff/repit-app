## Description:
This story enables the RepIt User to create and manage their demographic profile data, including personal information and profile media.

## User:
RepIt User

## Benefit Hypothesis:
As a RepIt User, I would like to manage my demographic information so that I can personalize my profile and enhance my overall experience.

## Acceptance Criteria:

I will be satisfied when:
- The RepIt User can add and manage the following Demographic Data within the User Profile
-  - Name is a required field
   - Name can be entered manually by the user
   - Name can be automatically populated from a linked third-party account:
   -  - Google Account
      - Apple Account (this has been delayed until R2)
      - Facebook Account
- - Age is an optional field
  - - Age can be entered as an integer value by the user
    - Age must be a valid integer if entered
- - Weight is an optional field
  - Weight can be entered as an integer value by the user
  - Weight must be a valid integer if entered
  - User can select unit of measurement for Weight:
  - - lbs
    - kg
    - Weight unit must be selected if weight is entered
- - Profile Picture is an optional field that can be added through the following methods
  - Capture a photo
  - Select or upload a photo from the device
  - The selected image is stored under the user profile
- User can edit demographic data
- User can save updates to their profile
- Profile data persists across sessions
